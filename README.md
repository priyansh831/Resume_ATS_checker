Resume ATS Checker

A streamlined, free online tool built to analyze resumes against Applicant Tracking Systems (ATS) to help job seekers optimize their applications.

This project was built for the Digital Heroes Custom Software Developer Trial Task.

Features & StackCore Functionality:
 Parses and evaluates resume text to deliver real, actionable optimization feedback.

Tech Stack: React / Node.js (MERN stack subset) deployed on Vercel (Hobby Plan).
  
Built With: Developed inside Google Antigravity using the Claude model.

Cost: $0 spent, entirely leveraging free tiers.
